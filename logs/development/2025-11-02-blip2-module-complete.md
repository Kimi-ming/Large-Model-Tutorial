### 2. BLIP-2推理示例代码
**文件**: `code/01-model-evaluation/examples/blip2_inference.py`  
**规模**: 约750行代码  

**核心功能**:
- `BLIP2InferenceService`类
  - 支持4种模型配置
  - 自动设备选择
  - FP16/FP32支持
- 6个主要推理方法:
  - `generate_caption` - 图像描述
  - `visual_question_answering` - VQA
  - `batch_inference` - 批量推理
  - `extract_features` - 特征提取
  - `compute_similarity` - 相似度计算
  - `benchmark` - 性能测试
- 完整的命令行接口
- 5种任务模式

**使用示例**:
```bash
# 图像描述
python blip2_inference.py --task caption --image example.jpg

# 视觉问答
python blip2_inference.py --task vqa --image example.jpg --question "What is this?"

# 性能测试
python blip2_inference.py --task benchmark --image example.jpg
```
